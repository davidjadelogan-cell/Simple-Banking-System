
A simple Python banking application that allows users to:
- Deposit money
- Withdraw money
- Generate transaction summaries
- Saves transaction summaries to a text file

## Features
- Input validation
- File handling
- Transaction tracking

## Technologies
- Python
- File I/Or a valid input
